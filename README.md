# Allumette
Allumette game, studient project made in C with mini-libix (graphic library).

 - IA XOR unbreakable
 - player vs player
 - Nice HUD (help pannel, home made graphics)
 - double display (graphic & in shell)
 - dynamic modification of line in-game
