# Nim

Nim game, studient project made in C with Epitech's mini-libX (graphic library).

 - unbreakable XOR AI
 - player vs player
 - Nice HUD (help pannel, home made graphics)
 - double display (graphical, shell)
 - dynamic modification of game elements
